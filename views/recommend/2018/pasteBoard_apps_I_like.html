      好用的剪贴板工具

      yammyxing 10/21/2018 apps paste

      ctrl + c 和 ctrl + v 是电脑操作中最常用的功能，这应该没有人有异议吧。

      但可惜的是，剪贴板默认一次只能存一个，也就是说，系统默认只会记住最后复制的那个记录。聪明的你，应该想到剪贴板不应该只有这个有限的功能，怎么说也应该支持粘贴多个历史记录啊。

      没错，优秀的剪贴板应该支持管理你存在剪贴板里面的所有内容。

      # mac 上优秀的剪贴板应用

      1. Paste

      这款应用不仅支持粘贴历史内容，还可以修改、预览、清除样式等高级功能，另外支持快捷键唤出，有多好用，谁用谁知道。

      这款应用是收费应用，目前已经加入 setapp 这种订阅是软件模式。

      官网链接：https://pasteapp.me/

      1. copyClip

      在遇到 paste 之前，我一直在 mac 上用 copyClip 这款剪贴板应用，功能相对简单，但也能满足大部分使用需求了。

      直接在 app store 上就能搜到这款应用。

      下载链接：‎CopyClip - Clipboard History on the Mac App Store

      # windows 上的剪贴板应用

      作为一个跨平台的使用者，我偶尔也会使用 windows 来完成一些工作。

      clibor 就是一款我在 windows 上常用的剪贴板过工具。

      功能齐全，媲美 paste，用起来很顺心。

      下载链接：Clibor English version | amuns:code
