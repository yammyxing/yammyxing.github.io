                      mac 上我常用的工具app

                      yammyxing 6/28/2018 mac apps

                      mac 电脑已经用了快两年的时间，在这期间，不论是工作或者是学习，mac 都发挥了巨大的作用，这篇文章来推荐我常用的一些重要工具。

                      # surge

                      Surge 是一款web开发和代理工具，它首先是一个代理服务器，会接管全局的 http/https 请求，这样的好处是可以提高访问速度或者对请求的域名做过滤处理。

                      Surge Mac - Advanced Web Debugging Proxy for Mac & iOS

                      # bear

                      iOS 和 macOS 平台最好用的 markdown 笔记应用，没有之一。之前的文章有过介绍，欢迎阅读：这个笔记 app 为什么这么受欢迎？ 另外，自带的 notes 应用也不错，但由于暂时还不支持 markdown，不作为我的首选。

                      Bear - Notes for iPhone, iPad and Mac

                      # The Unarchiver

                      压缩和解压工具，mac 自带的压缩工具本来没有问题，但如果是来自于 win 系统的压缩文件，在 mac 上用默认压缩工具打开会出错。

                      The Unarchiver | Top Free Unarchiving Software for macOS

                      # AppClear

                      绿色清理工具，清理文件更彻底。

                      AppCleaner 3.4 free download for Mac | MacUpdate

                      # spectacle

                      优秀的分屏工具，快速将 app 页面分屏，从而提高工作和学习效率。

                      Spectacle

                      # iPic

                      最好用的上传图片的工具，支持 markdown 链接，配合用 markdown 写作很棒，有默认的图床，也可以自己设置图床。

                      iPic - Easily Upload Pictures | Toolinbox

                      # kantu

                      这是一款腾讯出品的优质看图工具，比自带的 photos 应用要好用，可以帮助我们轻轻松松的管理电脑上的图片或视频。

                      Mac App Store 上的“看图-轻松找照片”

                      # vscode

                      作为一个程序员，编辑器必不可少，vscode 作为轻量级的编辑器，同时集成了 ide 的源代码管理、调试、插件库的功能，值得一试，个人感觉写 JavaScript 特别好用。

                      Download Visual Studio Code - Mac, Linux, Windows

                      # transmission

                      macOS 系统上的下载工具很多，transmission 是其中一款支持 BT 协议的开源软件，优点是下载速度快、界面简介好看。功能上跟迅雷类似，但不收费，觉得好可以考虑捐赠。

                      Transmission

                      # snip

                      优质的截图工具，腾讯出品，优点是可以滚动截屏。微信电脑版或者 qq 都有截图功能，也可以考虑。

                      Snip
